# 戰爭的四大轉折點

## 轉折點 I: ChatGPT 效應 (2022.11)
- **事件**: OpenAI 發布 ChatGPT。
- **影響**: 驗證了 RLHF 的有效性，將 LLM 從 "研究品" 變成 "產品"。

## 轉折點 II: Llama 開源 (2023.02/2023.07)
- **事件**: Meta 釋出 Llama 系列。
- **影響**: 催生了羊駝生態 (Alpaca, Vicuna)，打破了 OpenAI 的壟斷。

## 轉折點 III: Gemini 2.0 定義 Agentic Era (2024.12)
- **事件**: Demis Hassabis 發布 Gemini 2.0，宣告 Agentic 時代來臨。
- **影響**: 確立了 "Native Multimodal" 與 "Agentic Capabilities" 為新標準，預示了 MoE 架構的必要性。

## 轉折點 IV: MoE 的全面勝利與 Llama 4 轉向 (2025.04)
- **事件**: Meta 發布 Llama 4，放棄密集型架構，全面轉向 MoE。
- **數據**: DeepSeek V3 ($5.5M) 和 Kimi K2 ($4.6M) 的訓練成本震驚世界。
- **影響**: 證明了 "Architecture > Scale"。OpenAI 的 "Stargate" 計畫瞬間顯得過時。推理成本的崩塌使得 AI Agent 得以大規模普及。
