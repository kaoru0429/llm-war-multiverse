# OpenAI: 星門 (Stargate) 與末日困境

當全世界都在轉向高效的 MoE 架構時，OpenAI 似乎陷入了對「規模」的執著。

## 1. 星門計畫 (Project Stargate)

**最後的瘋狂？**
- **總投資**: **$500 Billion** (5000 億美元)。
- **能源需求**: 10 Gigawatts (相當於紐約市的用電量)。
- **目標**: 2029 年建成，旨在透過暴力算力實現 AGI。

## 2. 成本的荒謬對比

MoE 陣營對星門計畫的無情嘲諷：

| 項目 | OpenAI 星門 | MoE 陣營 (DeepSeek/Kimi) | 差異 |
| :-- | :-- | :-- | :-- |
| **預算** | $500,000,000,000 | ~$5,000,000 (訓練一個頂級模型) | **90,909 倍** |
| **時間** | 2029 年完工 | 明年就能迭代下一代 | 速度差異 |
| **哲學** | 力大磚飛 (Scale is all you need) | 巧勁致勝 (Architecture matters) | 路線分歧 |

**"我們用星門 1% 的預算，可以幫你們蓋 100 個頂級模型還有找。"**

## 3. GPT-5 的架構猶豫

傳聞指出 GPT-5 最終也被迫採用了 **Novel Mixture-of-Experts (MoE)** 架構，因為：
1. **高質量數據耗盡**: Public data 已被榨乾。
2. **成本壓力**: 如果不轉向 MoE，推理成本將無法與 DeepSeek (便宜 50 倍) 競爭。

## 4. 人才流失危機
OpenAI 變成了它最初反對的樣子：
- 11 位創辦人僅剩 2 位。
- 關鍵安全研究員 (Ilya, Jan Leike, John Schulman) 出走 Anthropic 或自立門戶。
- 價值觀從 "Safe AGI" 轉向 "Product Speed"。

OpenAI 依然強大，但在 2025 年，它不再是唯一的定義者。它的「星門」可能是通往 AGI 的大門，也可能是密集型架構時代的墓碑。
