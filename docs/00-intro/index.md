## 2. 核心衝突
- **Dense vs MoE**: 關於模型效率與擴展性的路線之爭。
- **Closed vs Open**: OpenAI/Google 的閉源壁壘 vs Meta Llama/Mistral 的開源突圍。
- **Human vs Data**: 合成數據 (Synthetic Data) 是否能取代人類標註？

## 3. 戰爭分期
- **第一階段：震驚與畏懼 (2022.11 - 2023.03)**
  - ChatGPT 發布，GPT-4 登場。
- **第二階段：百模大戰 (2023.04 - 2024.06)**
  - 開源模型爆發，Google Gemini 反擊，Claude 崛起。
- **第三階段：架構收斂與 AGI 曙光 (2024.07 - 2025.11)**
  - 稀疏化架構成為主流，推理成本大幅下降，Agent 系統成熟。
