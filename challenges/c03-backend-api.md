# 挑戰 03：LLM 戰爭數據 GraphQL API

## 目標
提供一個統一的 GraphQL 介面，供前端與其他服務查詢戰爭歷史數據。

## 需求
1. 定義 Schema (`Model`, `Company`, `Event`, `Person`)。
2. 實作 Resolver 邏輯。
3. 提供高效率的查詢 (支援 nested queries)。
4. 包含完整的 API 文檔與 Playground。

## 交付物
- GraphQL Schema (.graphql)
- 伺服器原始碼
- API 測試案例
