# 挑戰 02：戰爭歷史互動式網頁

## 目標
建立一個沈浸式的網頁，展示第一次 LLM 架構戰爭的關鍵節點與勢力消長。

## 需求
1. 使用現代前端框架 (React/Vue/Svelte)。
2. 實作互動式時間軸。
3. 勢力地圖或網絡圖，展示 OpenAI, Google, Anthropic, Meta 等公司的關係與人才流動。
4. 響應式設計，支援移動端。
