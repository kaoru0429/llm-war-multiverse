# 挑戰 05：LLM 發展策略模擬遊戲

## 目標
開發一款策略遊戲，讓玩家扮演 AI 實驗室負責人，在資源有限的情況下發展 AGI。

## 需求
1. 核心循環：獲取算力 -> 招募人才 -> 訓練模型 -> 發布產品 -> 獲取資金。
2. 隨機事件：監管打擊、人才出走、技術突破。
3. 科技樹系統：Transformer, MoE, RLHF, Chain-of-Thought。
4. 競爭對手 AI 邏輯。

- 遊戲可執行檔或網頁連結
- 設計文檔
