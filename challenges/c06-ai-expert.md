# 挑戰 06：戰爭歷史專家模型

## 目標
訓練或微調一個小型模型，使其成為「第一次 LLM 架構戰爭」的領域專家。

## 需求
1. 構建專屬數據集（基於本專案的 docs/ 內容）。
2. 使用 RAG (Retrieval-Augmented Generation) 或 Fine-tuning 技術。
3. 能準確回答關於具體事件、時間點、技術細節的問題。
4. 減少幻覺，提供引用來源。

## 交付物
- 模型權重 (LoRA) 或 RAG 系統代碼
- 評測報告
- 演示 Demo
