# 挑戰 01：第一次 LLM 架構戰爭數據科學分析

## 目標
分析 2022-2025 年間各大模型（GPT-4, Gemini, Claude, Llama 等）的參數規模、訓練算力、基準測試（MMLU, HumanEval）的演變趨勢。

## 需求
1. 收集公開的論文與技術報告數據。
2. 使用 Python (Pandas, Matplotlib/Seaborn) 進行數據清洗與視覺化。
3. 產出至少 5 張關鍵圖表：
   - 參數量隨時間的變化
   - 訓練成本估算曲線
   - 基準測試得分競賽圖
   - 上下文窗口增長趨勢

## 交付物
- Jupyter Notebook (`analysis.ipynb`)
- 數據集 CSV
- 分析報告 Markdown
